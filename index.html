Alur penggunaan
Cara kerjanya sederhana
1
Download & pasang extension
Install Chrome Extension dari tautan di bawah. Extension langsung aktif dan mulai memantau URL yang kamu kunjungi.
2
Browsing seperti biasa
Extension bekerja di background — setiap URL otomatis dianalisis. Badge ✓ atau ! muncul di icon extension sebagai indikator.
3
Lihat dashboard riwayat
Klik tombol "Lihat Dashboard" di popup extension, atau buka halaman Dashboard di web app ini untuk melihat visualisasi lengkap riwayat scan kamu.
4
Deteksi manual jika perlu
Masukkan URL apapun di kolom scan — baik di popup extension maupun di web app ini — untuk mendapatkan hasil analisis instan.
